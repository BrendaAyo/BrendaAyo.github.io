Lisbon Toursim WebApp


The application demonstration can be viewed HERE, with the GitHub repo available HERE.


Chamodi Boyagoda and I built a Location based web application for Tourism in Lisbon for our Group Project Seminar on Programming and Analysis final project. The web application provides comprehensive information on and assistance in one’s travel-related decision making while in Lisbon in order to enrich their tourist experience, accumulating points of interest, attractions based on user’s location and desired number of points to visit, with location and routing support.


The web application is based on open source data from OpenStreetMap and open source software i.e. Python, PostgreSQL, GeoDjango, leaflet and QGIS. The choice to use python and GeoDjango web framework for this project is based on the fact that GeoDjango can be used to create location-based web applications for any use and its object-relational mapping (ORM) is compatible with several databases. OpenStreetMap-based instruments were used for geographic information processing and map rendering with the additional information on the map like transportation means, restaurants etc. within a particular vicinity.


My work on this project focused mainly on database development, back-end data processing and configuring the interactive map. On the back-end, the application uses PostgreSQL-Deployed databases of tourist attraction locations in Lisbon and their complementing attributes. Data processing steps are performed using PostGIS and Python packages. Starting from a Geodjango template, the front-end consists of interactive visualizations and forms created with Leaflet, Mapbox and OpenLayers library and JavaScript libraries.
