Classifying and Monitoring Informal Settlements using Open GeoData


The full documentation for this research can be viewed HERE, with the GitHub repo available HERE.


This study demonstrated how OSM data and sentinel-2 imagery can be integrated for classifying and monitoring the spatio-temporal growth of informal settlements, employing Geospatial techniques and Artificial Intelligence. The identification and monitoring of informal settlements in urban areas is an important step in developing and implementing pro-poor urban policies. Understanding when, where and who lives inside informal settlements is critical to efforts to improve their resilience.


Three building feature geometrical characteristics of were derived and used to cluster and classify informal areas. Preprocessing of OSM data and Satellite Imagery through data cleaning and preparation suitable for Analysis. The OSM Building polygons were enrichment with Feature geometric and topologic characteristics (size, shape and Distance to nearest Neighbour). Hotspot clustering analysis on building features training dataset to cluster Informal settlement buildings from the formal ones. Test and select the best classifier based on accuracy to predict informal areas in a city based on similarity context of buildings outline characteristics.


The resultant informal regions in Kampala were used with Sentinel-2 imagery tiles to investigate the spatio- temporal changes in informal areas using Convolutional Neural Networks (CNNs). This stage involved slicing the images into tiles and using the predicted Informal settlement regions from the step above to generate training set on image tiles of corresponding date period. A Convolutional Neural Networks Model is trained to identify Image tiles that belong to Informal settlements. The trained model is then used to predict Informal settlement image tiles for other temporal datasets.


Results from Optimized Hot Spot Analysis and Random Forest Classification show that Informal regions can be mapped based on building outline characteristics. An accuracy of 90.3% was achieved when an optimally trained CNN was executed on a test set of 2019 satellite image tiles. Predictions of informality from new datasets for the years 2016 and 2017 provided promising results on combining different open source geospatial datasets to identify, classify and monitor informal settlements.
